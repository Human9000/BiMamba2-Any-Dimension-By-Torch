# BiMamba2 for any dimension
支持任意维度的双向Mamba2的纯torch模块

## 支持的功能
- ✅ 1d数据(batch,channel,length)
- ✅ 2d数据(batch,channel,height,width)
- ✅ 3d数据(batch,channel,deep,height,width)
- ✅ Nd数据(batch,channel,*size)
